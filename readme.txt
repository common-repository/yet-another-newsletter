A simple Wordpress Newsletter that works ! (Widget include)

== Description ==

With this plugin, you can send newsletters ! (magic)
But the best is to send newsletter to your public so there is a simple widget to do that ! (personnalize it within the code)
Want to remove subscribers ? You can !
Want to change the "From:" field ? You can !
Bored to write HTML to send mails ? You now can send a post !
Bored by ads… ? There are no ad (and use Adblock Plus for other websites).

Now, download this plugin and use it ;).

REALLY simple newsletter plugin, one file, use the php mail() function,
you can easily edit it.

== Installation ==

Via the search function in wordpress or manually :

1. Extract `yet-another-plugin.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the `Options → Newletter` to personalize it
1. Go to your widget page to add the newsletter widget
